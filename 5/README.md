# **Horloge connectée avec Raspberry Pi Pico W et Servo Moteur**#
Description du projet

Ce projet consiste à créer une horloge connectée utilisant un Raspberry Pi Pico W et un servo moteur.
Le Pico W se connecte à Internet pour récupérer l’heure actuelle via une API (par exemple ntptime), puis contrôle un servo moteur dont l’angle varie en fonction de l’heure.

L’objectif est de matérialiser l’heure en positionnant le servo à l’angle correspondant à la position d’une aiguille d’horloge.
